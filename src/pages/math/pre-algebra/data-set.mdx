A data set in mathematics is a collection of related data points or values, often used in statistics to analyze and interpret information.

---

- **Median**: The middle value in a data set.
- **Mean**: The average of all values in a data set.
- **Mode**: The most frequent value in a data set.
- **Range**: The difference between the largest and smallest values in a data set.

---
  
## Examples

### Median

$\{1,1,3,3,3,\textcolor{#7932ff}{6},7,8,8,9,9\}\\$

### Mean

$\frac{2+2+4+5+8+12}{6} = \frac{33}{6} = \textcolor{#7932ff}{5.5}$
> $\frac{\text{Total of all the numbers}}{\text{Count of the numbers}}$ 

### Mode

$\{2,3,3,5,\textcolor{#7932ff}{7},\textcolor{#7932ff}{7},\textcolor{#7932ff}{7},\textcolor{#7932ff}{7},9,13\}$

### Range
$
\{4,5,6,8,15,17\}\\

$
> $\small\text{largest number} - \text{smallest number}$
