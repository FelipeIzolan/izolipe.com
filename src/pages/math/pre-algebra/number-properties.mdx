Number properties refer to rules that you can apply to **real numbers** in mathematical operations.

---

## Commutative Property

<Image src={Image1} alt="commutative property addition visual demonstration" /><br/>
<Image src={Image2} alt="commutative property multiplication visual demonstration" />

The commutative property that applies to **addition and multiplication** states that **the order in which you add or multiply numbers does not affect the result**.

---

## Associative Property

<Image src={Image3} alt="associative property addition visual demonstration" /><br/>
<Image src={Image4} alt="associative property multiplication visual demonstration" />

The associative property that applies to **addition and multiplication** states that **the grouping of numbers does not affect the result**.

---

## Distributive Property

<Image src={Image5} alt="distributive property visual demonstration" />

The distributive property that applies to **multiplication using parentheses** states that you can **distribute the multiplication across each term inside the parentheses**.
