## O Básico

$\log_b{n}$

> O $b$ é a **base**.\
> O $n$ é o **argumento**.

$\log_b{n} = x \iff b^x = n$

> O logaritmo é uma função relacionada

$\log x = \log_{10} x$

> Quando a base não é explícita, a base é $10$.\
> ⚠️ Isso pode ser relativo.

---

## Logaritmo Natural (ou Neperiano)

$\log_e$

O logaritmo natural é um logaritmo que usa como base o número euler ($e$).

$\ln = \log_e$

Podemos usar $\ln$ como uma notação para $\log_e$.

> Só ignore isso por agora, é algo mais "avançado".
