mi (short for minimal) is my personal collection of projects, built around the KISS and suckless principles.

---

## Projects

- [**mi.css**](https://github.com/FelipeIzolan/mi.css) - minimal website cascading style sheets. (~1kb)
- [**mi.nvim**](https://github.com/FelipeIzolan/mi.nvim) - minimal neovim configuration concentrated in a single file.
- [**mi.void**](https://github.com/FelipeIzolan/mi.void) - minimal void linux setup. (wayland + hyprland)
- [**mi.astro**](https://github.com/FelipeIzolan/mi.astro) - minimal astrojs theme for contents. 

---

## A note about 'minimal'

Of course, what counts as a minimal setup, configuration, or software can be subjective. So this project reflects what I personally consider a minimal usable.
